# SCMDB Quest Recipe Patcher

Патчер для русской локализации Star Citizen. Добавляет:

- метку `[ЧЕРТЁЖ]` в название контрактов, где есть награды-чертежи;
- список доступных чертежей в описание контракта;
- данные берутся из актуальной базы [SCMDB](https://scmdb.net/).

## Обязательное требование

Сначала должен быть установлен русский перевод Star Citizen.

Рекомендуемый русификатор: [RuSC](https://www.expanseunion.com/sc/locru).

Обычный порядок такой:

1. Обновить Star Citizen.
2. Запустить русификатор RuSC, чтобы он создал свежий `global.ini`.
3. Запустить этот патчер.

Патчер не заменяет русификатор. Он только дополняет уже готовый файл локализации.

## Что изменяется

Патчер меняет только один файл:

```text
StarCitizen\LIVE\data\Localization\korean_(south_korea)\global.ini
```

Он не изменяет:

- архивы игры;
- исполняемые файлы;
- EasyAntiCheat;
- сохранения;
- игровые механики, баланс, награды или аккаунт.

Это текстовый патч локализации.

## Как запустить

Самый простой вариант:

1. Запустите `SCMDB_Quest_Recipe_Patcher.bat`.
2. Когда программа спросит путь к `StarCitizen\LIVE`, укажите путь, например:

```text
C:\Games\StarCitizen\LIVE
```

Вариант через PowerShell:

```powershell
.\SCMDB_Quest_Recipe_Patcher.ps1 "C:\Games\StarCitizen\LIVE"
```

Проверка без изменения файла:

```powershell
.\SCMDB_Quest_Recipe_Patcher.ps1 "C:\Games\StarCitizen\LIVE" -DryRun
```

## Что будет в игре

В списке контрактов у заданий с наградами-чертежами появится префикс:

```text
[ЧЕРТЁЖ]
```

Внизу описания контракта появится блок:

```text
Доступные чертежи (SCMDB)
- ...
```

Если один и тот же текст описания используется несколькими вариантами миссии с разными наградами, патчер пишет:

```text
Возможные чертежи (SCMDB)
```

В таком случае список объединён по известным вариантам этой миссии.

## Повторный запуск

Патчер можно запускать повторно. Он сначала убирает свои старые метки и блоки, затем ставит свежие данные из SCMDB.

Это полезно после:

- обновления Star Citizen;
- обновления русификатора;
- обновления базы SCMDB.

## Резервные копии и отчёты

Перед реальной записью патчер создаёт backup рядом с собой:

```text
backups\global.ini.YYYYMMDD-HHMMSS.scmdb-recipes.bak
```

Отчёты о запуске пишутся в:

```text
reports\
```

Эти папки не нужно добавлять в git.

## Настройки

Можно поменять метку в названии контракта:

```powershell
.\SCMDB_Quest_Recipe_Patcher.ps1 "C:\Games\StarCitizen\LIVE" -TitleMarker "[BP]"
```

Можно указать конкретный `global.ini`:

```powershell
.\SCMDB_Quest_Recipe_Patcher.ps1 -GlobalIniPath "C:\Games\StarCitizen\LIVE\data\Localization\korean_(south_korea)\global.ini"
```
