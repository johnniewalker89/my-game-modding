# SC Mod Launcher

Лаунчер для установки RuSC и безопасных подсказок Star Citizen через файл `global.ini`.

Лаунчер не меняет архивы игры, не трогает память, не ставит оверлеи и перед применением делает backup.

## Как установить

1. Скачайте релизный архив `SC_Mod_Launcher_2.1.4.zip`.
2. Откройте архив и извлеките из него папку `SC_Mod_Launcher` в удобное место.
3. Запускайте `SC_Mod_Launcher.exe` из этой папки.
4. Если Windows создала внешнюю папку вроде `SC_Mod_Launcher_2.1.4`, не используйте её как рабочее имя установки. Рабочая папка должна быть именно `SC_Mod_Launcher`.

## Первый запуск

1. Укажите путь к папке `StarCitizen\LIVE`.
2. Откройте вкладку `Русификатор`.
3. Установите или обновите RuSC.
4. Если RuSC уже был установлен вручную или старой версией лаунчера, лучше сначала удалить его через лаунчер и затем установить заново.
5. Нажмите `Проверить`.
6. Если лаунчер просит обновить cache, нажмите `Прогреть кэш`.
7. Выберите нужные модули и галки.
8. Нажмите `Применить в LIVE`.

## Что включают модули

`Майнинг и крафт` добавляет подсказки по ресурсам, крафту, переработке, скупке руды и ТТХ предметов.

`Квесты и рецепты` добавляет чертежи, репутацию, метки контрактов, подсветку выгодных фарм-квестов и Wikelo-подсказки.

## Как обновлять

На вкладке обновлений лаунчер проверяет GitHub Releases и ищет архив `SC_Mod_Launcher_*.zip`.

Если обновление найдено, используйте кнопку обновления в лаунчере. Пользовательские `config`, `backups` и backup обновления сохраняются.

## Как откатиться

Откройте вкладку `Backup`.

Можно восстановить последний backup `global.ini` или выбрать нужный файл вручную. Перед откатом лаунчер дополнительно сохранит текущее состояние LIVE.

## Если GitHub не открывается

Если не открывается `api.github.com`, RuSC можно скачать вручную как ZIP со страницы StarCitizenRu и установить кнопкой `Из ZIP` на вкладке `Русификатор`.
