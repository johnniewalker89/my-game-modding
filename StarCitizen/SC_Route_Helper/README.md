# SC Route Helper

Легкий helper для Star Citizen + zapret: ловит `30000` в `Game.log`, копит IP-кандидаты и создает новый zapret bat на основе уже рабочего bat-файла.

## Что делает

- Запоминает начало фрагмента `Game.log` по кнопке `Начать запись`.
- После ошибки разбирает только новый фрагмент лога.
- Находит `30000` с `CNetChannel::InactivityTimerCallback` и `remoteAddr`.
- Копит найденные IP в локальном списке.
- Создает отдельный `lists\ipset-starcitizen.txt` и новый `_SC_...bat`.

Helper не запускает zapret сам и не меняет файлы Star Citizen.

## Установка

1. Скачайте `SC_Route_Helper_v0.1.0.zip` на странице [Releases](https://github.com/johnniewalker89/my-game-modding/releases/tag/sc-route-helper-v0.1.0).
2. Распакуйте архив в любую удобную папку.
3. Запустите `SC_Route_Helper.bat`.
4. Выберите папку `StarCitizen\LIVE`.
5. Выберите исходный рабочий zapret `.bat`, когда будете создавать новый bat.

## Как пользоваться

1. Нажмите `Начать запись`.
2. Запустите Star Citizen и дождитесь ошибки `30000`.
3. Вернитесь в helper и нажмите `Остановить и разобрать`.
4. Нажмите `Создать bat`.
5. Остановите старый zapret и запустите новый `_SC_...bat` от администратора.

После новой `30000` повторите тот же цикл: список IP будет дополняться.
